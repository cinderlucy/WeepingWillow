Weeping Willow is a utility library for Roblox.

Existing Modules:
- LilacLINQ. Brings LINQ methods such as All, Any, Where, and Select to Luau
- MapleMap. State aware map with mutability, locking, unlocking, along with methods such as Join, Set, Copy, and DeepCopy.

Planned Modules:
- AcaciaArray
- SpruceSet
- TulipTimer
